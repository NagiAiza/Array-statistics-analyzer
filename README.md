## Author
 - **Name** : Alexis BOULIC
 - **Date** : 2026-06-07

## Description

This C program performs statistical analysis on an unsigned char array. It calculates the median, mean, maximum, and mimimum, sort in desending order the array and then display it on console.

## Contents
 * `stats.c`: Implementation of statistical and sorting functions.
 * `stats.h`: Function declarations and documentation.
 * `README.md`: project documentations.

## Usage
```bash
gcc -o stats.out stats.c
./stats.out
